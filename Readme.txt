各サンプルのSample*.pyの*の番号はそれぞれ以下の内容のサンプルであることを示す。

0.GoPiGoをセットアップし,VNC上で実行すると2秒ほど直進するGoPiGoのサンプル(python)

1.PCからVNC接続状態でキー入力によって動作するGoPiGoのサンプル

2.カメラを利用するサンプル
2.1 キー入力に応じて撮影して画像保存

3. ビジョンシステムを利用するサンプル
3.1 VSと通信してGoPiGo側でGoPiGoの座標を表示するサンプル

4. 評価プログラムのサンプル
4.1 撮影した画像をリアルタイムに評価するサンプル
->キー操作で撮影するたびにその画像の一定範囲内に一定の大きさで単色のマーカーが撮影できているかを評価し,OK/NGをコンソール

4.2 撮影した画像をまとめて評価するサンプル
->特定のフォルダ内の全画像を対象に一定範囲内に一定の大きさで単色のマーカーが撮影できているかを評価し,OK/NG枚数を出力
5．自動操縦に関するサンプル
5.2 特定の色を認識して撮影して画像保存
